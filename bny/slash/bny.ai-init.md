Run `bny ai init $ARGUMENTS` to bootstrap AI tool awareness.

Creates symlinks pointing each agent's context file to `bny/AGENTS.md`.
Supports: claude, gemini, agents, copilot, cursor, windsurf, kilocode,
auggie, roo, qwen, codebuddy, qoder, shai.

No args = all known agents. Pass specific names to init only those.
