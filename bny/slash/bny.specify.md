Run `bny specify $ARGUMENTS` to create a new feature workspace.

This creates a git branch and spec template at `specs/<branch>/spec.md`.

After running:
1. Review the generated spec template
2. Fill in user stories, requirements, and success criteria
3. The spec is the input for `bny plan`

Protocol: run `./dev/pre_flight` before starting work. See `bny/AGENTS.md`.
